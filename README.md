## Purpose
- Learn Git commands like `add`, `commit`, `push`, and `pull`
- Understand Git workflow from local repo to GitHub
- Practice creating branches and managing commits

## Files in this repository
- hello.txt → Example text file for practicing commits
- README.md → This file describing the project

## How to use
1. Clone the repository: `git clone https://github.com/aadyantchandra/git-practice.git`
2. Make changes to files
3. Stage changes: `git add .`
4. Commit changes: `git commit -m "Your message"`
5. Push changes: `git push`
